Tcl Stuff

This page collects some custom extensions and snippets I use in my Tcl projects.

C or mixed Tcl/C:

  • Critbit trees - COW-friendly dict replacement that supports efficient update for shared objects as well as additional operations.
  • Regex - a new regex package intended for UTF-8 mode.
  • Promises - explicit lazy evaluation.
  • Memoization - related to above.
  • Records - records with named fields.

Tcl-only code:

  • Parser combinators - DSL for writing parsers

Tcl customizations:

  • UTF-8 mode - operate on UTF-8 strings directly rather than convert to unicode (or UCS-2).
